#  AnimalCare – Application de Gestion de Clinique Vétérinaire

##  Description
AnimalCare est une application web développée dans le but de gérer les activités d’une clinique vétérinaire.  
Le système permet la gestion des utilisateurs, des vétérinaires, des propriétaires, des animaux, des rendez-vous et des horaires.  
L'application est basée sur une architecture MVC.



##  Objectifs du projet
- Fournir une gestion centralisée d’une clinique vétérinaire  
- Permettre la gestion des rendez-vous  
- Assurer la gestion des animaux et des propriétaires  
- Gérer les horaires des vétérinaires  
- Permettre la consultation du planning  

##  Utilisateurs du système
- Administrateur : gestion globale du système  
- Réceptionniste : gestion des rendez-vous  
- Vétérinaire : consultation du planning  
- Client : consultation et prise de rendez-vous  


##  Fonctionnalités principales
- Gestion des rendez-vous  
- Gestion des animaux  
- Gestion des propriétaires  
- Gestion des vétérinaires  
- Gestion des horaires  
- Consultation du planning  
- Authentification selon le rôle  


##  Architecture
Le projet suit l’architecture MVC :
- Models : gestion des données  
- Views : interface utilisateur  
- Controllers : logique métier  



## Design Patterns utilisés
Le projet utilise plusieurs design patterns afin d'améliorer la qualité du code :

- Singleton : gestion d'une instance unique globale  
- Observer : système de notifications  
- Strategy : gestion de comportements dynamiques  
- Decorator : ajout de fonctionnalités dynamiques
